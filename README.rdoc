=== 야구 중계 읽어주는 남자
네이버 프로야구 중계를 읽어주는 웹 어플리케이션
Demo: http://sports.news.naver.com/sports/new/main/index.nhn

=== Install
Requirements
* Python 2.7.3
* Django 1.4.1
Commands:
* python manage.py sql polls
* python manage.py syncdb
